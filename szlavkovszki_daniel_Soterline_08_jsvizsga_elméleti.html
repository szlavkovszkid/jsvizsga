Mikor használandó a JS egy weboldal felépítése során, hogyan használható?

Amikor tömbökkel, objectekkel, adatbázisokkal kell dolgozzunk vagy logikai műveletekkel. Ezen kívűs tudjuk a css és html részét is módosítani DOM manipulációval. Sorold fel, hogyan határozhatunk meg egy értéket és röviden definiáld őket.

Let helyi csak az adott futtatáson belűl lehet használni

var az egész js-en belűl érvényes és használható

const egyedi nem változó azonosító fix

Ismertesd a literálókat.

Integer (egész számokhoz használjuk)

Boolean (logikai ha valaminek csak két kimenetele lehet igaz vagy hamis)

String (sztring ezzel szöveget jelölűnk szabadon illetve tartalmazhat számokat is)

Array (tömb egy gyűtő amiben sok elemet/értéket el tudunk helyezni, több tömböt is egymásba lehet ágyazni)

Object (objektum egy gyűtő amiben sok elemet/értéket el tudunk helyezni, hasonlít a tömbhöz, de itt inkább json és adatbázis adatokhoz használjuk)

Miért érdemes függvényeket használni?

Mert azokat csak egyszer kell megírni utána function-ként bármikor vissza tudjuk hívni ahol szükségünk van rá. Akkár egy sima összeadást is.

Hogyan paraméterezhetünk fel egy függvényt?

Minden esetben szüksége elátnunk változókkal, amikbe behetesíthetjük az értéket.

Mi az a DOM?

Amikor js segítségével az eredeti html és css adatokat tartalmat, formát a js kódunk futtásával megváltoztatjuk újabb értékket adunk hozzá

Kicserélünk és visszaírunk értékeket anélkül, hogy hozzányúlnánk az eredeti html vagy css kódhoz a js futtatásával módosítjuk. Mire utasít a következő példa: document.getElementByClassName(“example”).innerText

Hogy vegye fel értéknek az example class névvel elátott szöveget.

Hogyan nyerhető ki egy form input mező értéke és hogyan jeleníted azt meg HTML-ben?

Egyedi id azonosítót adunk neki és az abba beírt értéket elmentjük egy változóba.

majd a vátozót DOM manupulációval visszaírjuk egy általunk meghatározott tag-be szintén egyedi értékkel.

var password = document.getElementById("password").value

document.getElementById("paros1").innerHTML = password;
